# Excel Sales Automation

Ferramenta local para automatizar consolidação de planilhas de vendas,
gerar subplanilhas, gráficos e insights automaticamente usando LLMs.

## Objetivo
Reduzir erros manuais e tempo gasto em análise de vendas.

## Stack
- Python
- Pandas
- LangChain / LangGraph
- FastAPI
- Docker
